selector in javascript : 
        (1) document.querySelector()
        (2) document.getElementById()
        (3) document.getElementsByClassName() = Get the HTML collection on which the method was invoked 
                                                with ClassName.
        (4) document.getElementsByName() = Gets a collection of objects based on the value of the NAME 
                                           or ID attribute.
        (5) document.getElementsByTagName() = Retrieves a collection of objects based on the specified 
                                              element name.
        (6) document.querySelectorAll() = Returns all element descendants of node that match selectors.

        to print value in browser used two property :
        (1) innerHTML = display message with HTML structure
        (2) innerText = display only HTML message
