there is two type of datatype

  • primitive
  • non-primitive

Primitive datatype

which is inbuilt in program.

  1. Number - for ex, 12 and 12,5
  2. String - for ex, "Hello World"
  3. Boolean - return true or false
  4. Null - null value
  5. Undefined - undefined value

Non-primitive datatype

  1. Array
  2. Object
  3. Regular Expression
