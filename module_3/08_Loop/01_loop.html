loop is count number of iteration.

    There are 3 type of loop :
    (1) while loop : It through a block of code as long as a specified condition is true.
        syntax : 
        while(condition){
            statments;
            increment/decrement;
        }
        
    (2) do...while loop : This loop will execute the code block once, before checking if the condition is true.
        syntax :
        do{
            statments;
            increment/decrement;
        }while(condition)

    (3) for loop
        syntax :
        for(declaration; condition; increment/decrement){
            statments;
        }

    (4) for/in : use with object
        syntax :
        for(key in object){
            statments
        }

    (5) for/of : use with array for count itration
        syntax :
        for(iterator of object){
            statments
        }
